# Pinterest Image Downloader

Парсер для автоматического скачивания изображений с Pinterest в полном размере.

## Возможности

- ✅ Скачивание изображений с досок Pinterest
- ✅ Скачивание изображений со страниц пользователей
- ✅ Автоматическое получение изображений в **полном размере** (originals)
- ✅ Автоматическая прокрутка страницы для загрузки всех изображений
- ✅ Уникальные имена файлов
- ✅ Пропуск уже скачанных изображений
- ✅ Работа на Windows

## Требования

- Python 3.7 или выше
- Google Chrome браузер (для Selenium)
- Интернет соединение

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Убедитесь, что у вас установлен Google Chrome. WebDriver для Chrome установится автоматически при первом запуске.

## Использование

### Базовое использование

Запустите скрипт:
```bash
python pinterest_parser.py
```

Программа попросит вас ввести:
1. **URL доски или страницы Pinterest** - например:
   - `https://www.pinterest.com/username/board-name/`
   - `https://www.pinterest.com/username/`

2. **Имя папки для сохранения** (необязательно) - по умолчанию `pinterest_images`

### Примеры URL

- Доска пользователя: `https://www.pinterest.com/johndoe/my-board/`
- Страница пользователя: `https://www.pinterest.com/johndoe/`
- Публичная доска: `https://www.pinterest.com/username/board-name/`

## Как это работает

1. Парсер открывает указанную страницу Pinterest в браузере Chrome
2. Автоматически прокручивает страницу для загрузки всех изображений (lazy loading)
3. Извлекает URL всех изображений
4. Преобразует URL в формат полного размера (originals)
5. Скачивает каждое изображение в указанную папку
6. Сохраняет изображения с уникальными именами

## Особенности

### Полный размер изображений

Парсер автоматически преобразует URL изображений для получения полного размера:
- Заменяет размеры в пути URL (например `/564x/` → `/originals/`)
- Удаляет параметры размера из query string
- Использует оригинальные изображения Pinterest

### Обработка ошибок

- Пропуск уже существующих файлов
- Обработка ошибок сети
- Логирование процесса скачивания
- Статистика успешных и неудачных загрузок

## Примечания

- Для работы требуется активное интернет соединение
- Процесс может занять время в зависимости от количества изображений
- Pinterest может ограничивать частоту запросов - парсер включает задержки между запросами
- Некоторые приватные доски могут требовать авторизации

## Устранение неполадок

### ChromeDriver не найден
WebDriver должен установиться автоматически. Если возникают проблемы, убедитесь что Chrome установлен и обновлен.

### Не скачиваются изображения
- Проверьте, что URL правильный и доступен
- Убедитесь, что доска/страница публичная
- Попробуйте открыть URL в браузере вручную

### Медленная работа
- Уменьшите количество прокруток в коде (параметр `max_scrolls`)
- Убедитесь в стабильном интернет соединении
