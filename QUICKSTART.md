# Быстрый старт - Pinterest Image Downloader GUI

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python pinterest_gui.py
```

## Первое использование

1. Введите URL доски Pinterest (например: `https://ru.pinterest.com/username/board-name/`)
2. Укажите количество изображений (например: `10` для первых 10 изображений, или `0` для всех)
3. Выберите папку для сохранения
4. Нажмите "Начать скачивание"

## Все 12 функций приложения

1. ✅ **Ограничение количества** - укажите сколько изображений скачать
2. ✅ **Выбор папки** - удобный выбор через диалог
3. ✅ **Прогресс-бар** - визуальный прогресс скачивания
4. ✅ **Лог действий** - подробный лог с временными метками
5. ✅ **Фильтр по размеру** - скачивайте только нужные размеры (в МБ)
6. ✅ **Выбор качества** - полный размер / средний / маленький
7. ✅ **Пауза/возобновление** - контроль процесса скачивания
8. ✅ **История** - сохранение всех скачанных досок
9. ✅ **Автопереименование** - понятные имена файлов
10. ✅ **Экспорт URL** - сохранение списка всех URL
11. ✅ **Настройки задержек** - контроль скорости работы
12. ✅ **Предпросмотр** - миниатюры найденных изображений

## Примеры использования

### Скачать первые 10 изображений
- Количество: `10`
- Качество: `Полный размер`
- Нажать "Начать скачивание"

### Скачать только большие файлы
- Количество: `0` (все)
- Размер файла: От `1` до `1000` МБ
- Начать скачивание

### Быстрое скачивание
- Задержка прокрутки: `1.0` сек
- Задержка скачивания: `0.1` сек
- Начать скачивание

### Экономия места
- Качество: `Средний` или `Маленький`
- Начать скачивание

## Советы

- Используйте историю для быстрого доступа к ранее скачанным доскам
- Экспортируйте URL для резервного копирования
- Предпросмотр показывает первые 50 найденных изображений
- Приложение автоматически останавливается перед разделом "Похожие пины"
