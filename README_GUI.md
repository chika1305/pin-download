# Pinterest Image Downloader - GUI версия

Windows приложение с графическим интерфейсом для скачивания изображений с Pinterest.

## Возможности

### Основные функции:
1. ✅ **Ограничение количества изображений** - укажите сколько фотографий скачать (например, 10 с начала доски)
2. ✅ **Выбор папки для сохранения** - удобный выбор папки через диалог
3. ✅ **Прогресс-бар и статистика** - визуальное отображение прогресса скачивания
4. ✅ **Лог действий** - подробный лог всех операций с временными метками

### Дополнительные функции:
5. ✅ **Фильтр по размеру файла** - скачивайте только изображения определенного размера (в МБ)
6. ✅ **Выбор качества изображений** - полный размер, средний или маленький
7. ✅ **Пауза/возобновление** - возможность приостановить и продолжить скачивание
8. ✅ **История скачиваний** - сохранение истории всех скачанных досок
9. ✅ **Автоматическое переименование** - файлы автоматически получают понятные имена
10. ✅ **Экспорт списка URL** - сохранение всех найденных URL в текстовый файл
11. ✅ **Настройки задержек** - настройка скорости прокрутки и скачивания
12. ✅ **Открытие папки** - быстрый доступ к скачанным изображениям

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Убедитесь, что установлен Google Chrome

## Запуск

Запустите GUI приложение:
```bash
python pinterest_gui.py
```

## Использование

1. **Введите URL** доски или страницы Pinterest
2. **Укажите количество** изображений для скачивания (0 = все)
3. **Выберите папку** для сохранения
4. **Настройте параметры** (качество, размер, задержки)
5. **Нажмите "Начать скачивание"**

### Примеры настроек:

- **Скачать первые 10 изображений**: Установите "Количество изображений" = 10
- **Только большие файлы**: Установите "От 1 МБ до 1000 МБ"
- **Быстрое скачивание**: Уменьшите задержки до минимума
- **Экономия места**: Выберите качество "Средний" или "Маленький"

## Интерфейс

### Основные настройки
- URL доски/страницы
- Количество изображений (0 = все)
- Папка для сохранения
- Кнопка истории

### Дополнительные настройки
- Качество изображений (полный/средний/маленький)
- Фильтр по размеру файла
- Автоматическое переименование
- Задержки прокрутки и скачивания

### Управление
- **Начать скачивание** - запуск процесса
- **Пауза** - приостановка/возобновление
- **Остановить** - полная остановка

### Прогресс
- Текущий статус
- Прогресс-бар
- Статистика (найдено/скачано/ошибок/пропущено)

### Лог
- Подробный лог всех действий
- Кнопки: Очистить, Экспорт URL, Открыть папку

## Особенности

- **Остановка на границе доски** - парсер автоматически останавливается перед разделом "Похожие пины"
- **Пропуск существующих файлов** - не скачивает уже загруженные изображения
- **Обработка ошибок** - продолжает работу при ошибках отдельных файлов
- **Многопоточность** - GUI не зависает во время скачивания

## История

Все скачивания сохраняются в файл `download_history.json`. Вы можете:
- Просмотреть историю через кнопку "История"
- Быстро использовать URL из истории
- Видеть количество скачанных изображений

## Экспорт URL

После скачивания вы можете экспортировать все найденные URL в текстовый файл для дальнейшего использования.

## Примечания

- При первом запуске автоматически установится ChromeDriver
- Для работы требуется активное интернет соединение
- Некоторые приватные доски могут требовать авторизации в браузере
- Рекомендуется не ставить слишком маленькие задержки, чтобы не получить блокировку

## Устранение неполадок

### Приложение не запускается
- Убедитесь, что установлен Python 3.7+
- Проверьте установку всех зависимостей: `pip install -r requirements.txt`

### ChromeDriver ошибки
- Обновите Chrome до последней версии
- Перезапустите приложение (ChromeDriver установится автоматически)

### Не скачиваются изображения
- Проверьте URL доски
- Убедитесь, что доска публичная
- Проверьте интернет соединение
